오늘 배운것 복습하는 페이지

오늘배운 html 기초 용어들

글을 나눠주는 부분에 대하여


hr탭 : 가로줄을 그어주어 문단을 나눠주는 기능이다
ex : 글을 나눠주는 부분에 대하여 와 html 기초 용어들 사이의 가로줄

br탭 : 엔터기능
ex : hr부분 ex와 br 사이의 공백

blockquote탭 : 앞부분에 여유공간을 내주는것
ex : ex앞부분의 여유공간을 준것

p탭 : 글자가있는부분만이아닌 전체적인 문단으로 따로 쳐준다

p탭 예문

ex : 윗 문장이 예시이다

h탭 : h1~h6까지있으며 제목에주로쓰인다
ex : h1은 오늘 배운것 복습하는 페이지, h2글을 나눠주는 부분에대하여 가 예시이다

글자 모양에 관련된모양


strong탭, b탭 : 글자를 진하게 해준다
차이점 : strong은 읽어주는 프로그렘이 좀더 큰소라로 읽어준다 그에비해 b탭은 글자만 진해진다

em탭, i탭 : 글자가 기울어진다
차이점 : 읽어주는 프로그렘에서 또 차이가 있다
